# 介绍

## 什么是基金宝？

基金宝是一款开源的基金实时估值查看工具，旨在为基金投资者提供简洁、高效的基金跟踪体验。

## 主要特点

### 🚀 实时数据

- 交易时间内实时刷新估值
- 秒级更新，不错过任何波动

### 📱 Android 原生

- 原生 Android APP 体验
- 流畅的动画和交互
- 支持后台运行和通知

### 🔒 隐私安全

- 数据本地存储
- 无需注册登录
- 不收集任何个人信息

### 💯 完全免费

- 无广告
- 无付费功能
- 开源透明

## 适用人群

- 基金定投用户
- 主动管理型投资者
- 需要实时跟踪基金净值的用户
- 喜欢开源软件的技术爱好者

## 技术栈

| 技术 | 说明 |
|------|------|
| Vue 3 | 前端框架 |
| TypeScript | 类型安全 |
| Vite | 构建工具 |
| Vant | UI 组件库 |
| Pinia | 状态管理 |
| Capacitor | 移动端打包 |
| Canvas API | 图表绘制 |
